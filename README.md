# campus-e-voting-system

Campus E-Voting System is a web-based student election platform designed to facilitate campus elections efficiently. The project follows the Software Development Life Cycle (SDLC) phases to ensure a structured approach:

- **Communication:** Requirements were gathered through discussions with stakeholders to understand the needs of students and administrators.
- **Planning:** A detailed project plan was created outlining timelines, resources, and milestones.
- **Design:** The system architecture and user interfaces were designed focusing on usability and security.
- **Development:** Frontend was developed using HTML, CSS, and JavaScript, while backend services utilized Firebase Authentication and Realtime Database.
- **Testing:** Rigorous testing was conducted to validate functionality, performance, and security.
- **Deployment:** The application was deployed on Netlify for reliable and scalable hosting.
- **Maintenance:** Ongoing support and updates are planned to ensure system reliability and incorporate user feedback.

This structured approach ensures a robust and user-friendly platform for campus election management.
